# 敌人、Boss、关卡与难度

## 当前刷怪规则

- 怪物只在地图边框内刷新。
- 刷怪以波次为单位，不是一只只单独刷。
- 怪物出现前会在刷新点显示红色警告。
- 无尽模式按时间推进怪物组合，并按轮次提升敌人属性。

## 敌人方向

- 追击怪：基础近身压力。
- 远程怪：弹道压力。
- 加速怪：走位压力。
- 冲刺怪：突然贴脸压力。
- 精英 / 小 Boss：阶段性奖励和压力节点。

## Boss 方向

- Boss 不只是血量更高，而要通过阶段技能制造压力。
- Boss 弹道速度不跟随普通远程怪的削弱。
- 最终 Boss 奖励走最终核心奖励流程。
- 小 Boss 奖励走装备/训练等当前奖励流程。

## 难度 Profile

当前难度入口：

```text
easy / normal / hard / hell
```

难度不只修改血量，还影响：

- 刷怪间隔。
- 敌人生命、伤害、速度。
- 危险怪权重。
- 特殊怪攻击频率。
- Boss 压力。
- 运行时上限。

## 性能保护

- `scripts/game/difficulty_profile.gd` 携带敌人、弹幕和临时特效上限。
- `scripts/game/performance_guard.gd` 控制高密度场景下的运行时压力。
- 高频扫描行为应按间隔采样，不要每帧遍历大组。
- 大量命中判定应优先考虑批处理、空间分区或节流。

## 调参原则

- 简单难度用于学习和试祝福。
- 普通难度用于基准平衡。
- 困难难度提高危险组合出现率。
- 地狱难度提高多轴压力，但不应锁死唯一答案。
- 不建议通过大幅降低经验来制造难度，因为这会破坏升级和祝福节奏。
